# **App Name**: Stockseer

## Core Features:

- Ticker Input: Input field to accept stock ticker symbols.
- 5-Day Forecast: Use a tool to provide a 5-day stock forecast, using ticker as input.
- Data Visualization: Display of projected opening and closing prices for the next 5 trading days, as well as projected gain/loss.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to inspire confidence and stability in financial data.
- Background color: Light grey (#ECEFF1) to provide a clean, neutral backdrop.
- Accent color: Teal (#009688) to highlight key information and interactive elements.
- Body and headline font: 'Inter', a sans-serif, providing a modern, readable style appropriate for displaying numerical data.
- Data is clearly presented in a tabular format.
- Use subtle transitions when the forecast data is loaded to smoothly reveal the information.
